ELEMENTOS HTML

H1

El H1 es una etiqueta que se utiliza para identificar aquellas frases que conforman el título principal de un contenido de una página web.


P

Este elemento HTML define un párrafo. Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.


IMG

Para poner una imagen simple en una página web, utilizamos el elemento img. Se trata de un elemento vacío (lo que significa que no contiene texto o etiqueta de cierre) que requiere de por lo menos de un atributo para ser utilizado: src (a veces denominado por su nombre completo, source).


A

Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.


SRC

El atributo “src” es imprescindible para poder colocar una imagen. Este atributo es el que indica dónde se encuentra alojada la imagen que queremos mostrar.


STYLE

La etiqueta se utiliza para definir la información de estilo (CSS) de un documento. Dentro del style elemento, especifica cómo se deben representar los elementos HTML en un navegador.


HREF

El href atributo especifica la URL de la página a la que va el enlace.
